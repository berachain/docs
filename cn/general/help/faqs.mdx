<AccordionGroup>
  <Accordion title="Berachain 的性能指标如何？">
    Berachain 具有以下特性：

    - 出块时间：出块时间会有变化，请查看 [Berascan](https://berascan.com) 获取最新数据。
    - 每秒交易数（TPS）：会变化，可用以下公式估算：(区块 gas 上限 (30m) / 每笔交易平均 gas 上限) / 出块时间 (2s) = TPS。
    - 最终性：单槽最终性
  </Accordion>

  <Accordion title="什么是 DEX？">
    DEX 即去中心化交易所。您可以直接在链上买卖代币，而无需通过任何单一中心化服务。这意味着所有流动性都可以在链上直接查看，且由智能合约可验证地持有。DEX 让您可以直接从钱包兑换代币，并允许任何人发行自己的代币和提供流动性。
  </Accordion>

  <Accordion title="什么是 swap？">
    Swap 是将一种代币兑换成另一种的过程。根据您关注的代币，可以理解为买入或卖出。例如，若您想用 `$ETH` 买 `$BERA`，就是在用 `$ETH` 换 `$BERA`，本质上就是「卖出」`$ETH`、「买入」`$BERA`。
  </Accordion>

  <Accordion title="Swap 费用多少？">
    每笔 swap 都有费用，具体取决于池创建时设定的费率。常见费率为 0.05%、0.1%、0.3% 或 1%，但执行 swap 时请务必确认该池的费率是否合适。
  </Accordion>

  <Accordion title="什么是流动性？">
    流动性指可用于 swap 的代币数量。代币流动性越多，越容易进行 swap。
  </Accordion>

  <Accordion title="什么是流动性池？">
    流动性池是 2 个或更多代币的配对，流动性提供者将代币存入其中，使 DEX 用户可以在池内任意代币之间 swap。
  </Accordion>

  <Accordion title="什么是流动性提供者？">
    流动性提供者将代币存入流动性池，从池内 swap 产生的手续费中获得一部分。
  </Accordion>

  <Accordion title="什么是 APY？">
    APY 即年化收益率。在 BEX 池的语境下，指该池的当前 APY。APY 收益来自使用该池的每次 swap 所收取的手续费。
  </Accordion>

  <Accordion title="什么是 $HONEY？">
    `$HONEY` 是 Berachain 生态的原生稳定币。它是多抵押品支持的稳定币，在整个 Berachain 生态中使用。
  </Accordion>

  <Accordion title="铸造或销毁 $HONEY 是否收费？">
    为确保稳定，每次铸造和销毁 `$HONEY` 都会收取少量费用。此外，铸造和销毁需要交易，因此会产生少量 `$BERA` gas 费。
  </Accordion>

  <Accordion title="什么是 $BGT？">
    $BGT 是 Berachain 的质押与治理代币，用于通过流动性证明保障网络并赚取奖励，以及参与治理提案投票。
  </Accordion>

  <Accordion title="什么是验证者？">
    验证者可以指三件事：

    1. 验证交易、生产区块并与其他验证者达成共识的区块链节点
    2. 拥有并运营验证者节点的实体
    3. 上述 1 与 2 的结合，管理部分流动性证明与治理投票
  </Accordion>

  <Accordion title="为什么应该用 $BGT 提升验证者？">
    委托 `$BGT` 让您参与流动性证明并帮助保障网络。
  </Accordion>

  <Accordion title="为什么应该用 $BGT 提升验证者而不是销毁换 $BERA？">
    主要原因是奖励。通过流动性证明，您可以获得多种奖励：

    - 协议提供的[激励](/cn/general/proof-of-liquidity/incentives)的一部分，作为将 `$BGT` 排放导向这些协议奖励金库的回报
    - Berachain 核心 dApp 手续费的一部分，即 BEX 和 HoneySwap 的手续费
  </Accordion>

  <Accordion title="如何获得 $BGT？">
    当验证者将 `$BGT` 排放导向奖励金库时，您可通过奖励金库赚取 `$BGT`。详见[赚取 $BGT](/cn/general/tokens/bgt#earning-bgt)。
  </Accordion>

  <Accordion title="什么是治理？">
    治理是社区决定对 Berachain 协议进行哪些变更的过程，包括如何升级节点以及为链上各组件设置哪些参数。
  </Accordion>

  <Accordion title="在 BEX 向符合资格的池提供流动性后，如何获得 $BGT？">
    BEX 上每个符合条件（已白名单）的池都有对应的 LP 代币。向 BEX 池存入流动性后，池会按您的总贡献比例发放 LP 代币。您须将该 LP 代币质押到对应的奖励金库才有资格领取 `$BGT`。当验证者将 `$BGT` 排放导向奖励金库时，您会累积可领取的 `$BGT`。您需要再执行一次操作才能领取 `$BGT`——协议不会自动发送给您。
  </Accordion>

  <Accordion title="只有验证者才能对提案投票或创建提案吗？">
    只要持有规定最低数量的 `$BGT`，任何人都可以提出提案并对提案投票。
  </Accordion>

  <Accordion title="网络的实际质押代币是 $BERA 还是 $BGT？">
    - 验证者质押 `$BERA`
    - 网络激励以 `$BGT` 发放
  </Accordion>

  <Accordion title="仅用 $BERA 的验证者可以生产区块吗？奖励是什么？">
    可以。验证者只需在**最低 250,000、最高 10,000,000** 的范围内质押 `$BERA`，进入活跃集后即可提议区块。验证者获得的奖励以 `$BGT` 计。
  </Accordion>
</AccordionGroup>
