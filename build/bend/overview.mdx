<Note>
Bend is only available on Berachain Mainnet and has no plans to deploy to Bepolia Testnet.
</Note>

Bend is built on [Morpho V1](https://docs.morpho.org/build/earn/get-started). You can integrate smart contracts, oracles, flash loans, and new vaults and markets.

### Deployed contracts

See [Deployed Contracts](/build/bend/deployed-contracts) for all active contract addresses on supported networks.

### Deployed markets

See [Deployed Markets](/build/bend/deployed-markets) for live lending and borrowing markets, including utilization, liquidity, and metrics.

### Tooling

See [Onchain Tooling](/build/bend/onchain-bundlers) for programmatic and UI tooling to read and interact with Bend.

### Contract references

See [Contract References](/build/bend/contract-morpho) for function signatures, addresses, and deployed code to integrate or audit Bend contracts.
