<Note>
Bend is only available on Berachain Mainnet and has no plans to deploy to Bepolia Testnet.
</Note>

Bend is built on top of [Morpho V1](https://docs.morpho.org/build/earn/get-started), providing developers with powerful tools to integrate smart contracts, leverage oracles, utilize flash loans, curate configurations, and introduce new vaults and markets seamlessly within the ecosystem.

### Deployed contracts

Check out the [Deployed Contracts](/build/bend/deployed-contracts) section — it lists all active smart contracts, their addresses, and deployments across supported networks.

### Deployed markets

Check out the [Deployed Markets](/build/bend/deployed-markets) section — it showcases the live lending and borrowing markets, including utilization, liquidity, and performance metrics.

### Tooling

Explore [Onchain Tooling](/build/bend/onchain-bundlers) — this section provides basic tooling to interact with Bend's protocol data both programmatically and through user interfaces.

### Contract references

Visit the [Contract References](/build/bend/contract-morpho) section — learn about the available functions, contract addresses, and deployed code to integrate or audit Bend's smart contracts effectively.
