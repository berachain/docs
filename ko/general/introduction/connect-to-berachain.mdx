export const addNetworkMainnet = () => {
  if (typeof window === "undefined" || !window.ethereum) {
    console.warn("[AddNetwork] No wallet detected");
    return;
  }
  const params = {
    chainId: "0x138de",
    chainName: "Berachain",
    nativeCurrency: { name: "BERA Token", symbol: "BERA", decimals: 18 },
    rpcUrls: ["https://rpc.berachain.com/"],
    blockExplorerUrls: ["https://berascan.com/"],
  };
  console.log("[AddNetwork] add mainnet", params);
  window.ethereum.request({ method: "wallet_addEthereumChain", params: [params] }).then(r => console.log("[AddNetwork] result", r)).catch(e => console.error("[AddNetwork] error", e));
};

export const addNetworkBepolia = () => {
  if (typeof window === "undefined" || !window.ethereum) {
    console.warn("[AddNetwork] No wallet detected");
    return;
  }
  const params = {
    chainId: "0x138c5",
    chainName: "Berachain Bepolia",
    nativeCurrency: { name: "Testnet BERA Token", symbol: "BERA", decimals: 18 },
    rpcUrls: ["https://bepolia.rpc.berachain.com/"],
    blockExplorerUrls: ["https://testnet.berascan.com/"],
  };
  console.log("[AddNetwork] add bepolia (testnet)", params);
  window.ethereum.request({ method: "wallet_addEthereumChain", params: [params] }).then(r => console.log("[AddNetwork] result", r)).catch(e => console.error("[AddNetwork] error", e));
};

블록체인 지갑을 통해 Berachain의 자산에 접근할 수 있습니다. 지갑은 토큰 거래, NFT 구매, 게임 플레이 등에서 소유권을 증명할 수 있도록 개인 키를 생성하고 저장합니다.

## Berachain 메인넷 RPC

사용자 정의 RPC를 지원하는 모든 지갑에서 다음 값을 사용하세요. 원클릭으로 네트워크를 추가하거나 테이블에서 복사하세요:

<button type="button" onClick={() => addNetworkMainnet()} style={{ display: "inline-flex", alignItems: "center", justifyContent: "center", minWidth: "140px", height: "44px", padding: "0 16px", borderRadius: "8px", border: "2px solid #c96516", backgroundColor: "#E17726", color: "#1a1a1a", fontWeight: 600, cursor: "pointer" }}>Berachain 추가</button>

| 키                | 값                           |
| ----------------- | ---------------------------- |
| 네트워크          | `Berachain`                  |
| RPC URL           | `https://rpc.berachain.com/` |
| Chain ID          | `80094`                      |
| 통화 기호         | `BERA`                       |
| 블록 탐색기 URL   | `https://berascan.com/`      |

## 지원 지갑

사용자 정의 RPC를 지원하는 [EVM 기반 지갑](https://ethereum.org/en/wallets/find-wallet/)은 Berachain에서 사용할 수 있습니다. 예:

- [MetaMask](https://metamask.io)
- [Rabby](https://rabby.io)
- [Coinbase Wallet](https://www.coinbase.com/wallet)
- [Brave Wallet](https://brave.com/wallet/)
- [Frame](https://frame.sh)
- [Zerion](https://zerion.io/download?utm_source=berachain)

## Berachain 네트워크 수동 추가

위의 원클릭 버튼을 사용하지 않으려면 MetaMask에서 수동으로 네트워크를 추가하세요:

1. **네트워크 목록 열기** — MetaMask 왼쪽 상단의 드롭다운을 클릭하세요.

<Frame>
  <img src="/images/general/metamask-add-network-01.png" alt="MetaMask 네트워크 드롭다운 메뉴" />
</Frame>

2. **네트워크 추가** — 모달에서 `네트워크 추가`를 클릭하세요.

<Frame>
  <img src="/images/general/metamask-add-network-02.png" alt="모달의 MetaMask 네트워크 추가 버튼" />
</Frame>

3. **네트워크 수동 추가** — 목록 하단에서 `네트워크 수동 추가`를 클릭하세요.

<Frame>
  <img src="/images/general/metamask-add-network-03.png" alt="MetaMask 네트워크 수동 추가 옵션" />
</Frame>

4. **세부 정보 입력** — 위 [메인넷](#berachain-메인넷-rpc) 또는 [Bepolia 테스트넷](#berachain-bepolia-테스트넷-rpc) RPC 테이블의 값을 사용하세요.

5. **저장** — `저장`을 클릭하세요.

<Frame>
  <img src="/images/general/metamask-add-network-04.png" alt="Berachain 세부 정보가 있는 MetaMask 네트워크 구성 양식" />
</Frame>
